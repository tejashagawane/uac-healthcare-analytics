# 📊 UAC Healthcare Capacity Analytics Dashboard

## 🔍 Overview
This project analyzes system capacity and care load in the UAC (Unaccompanied Alien Children) program using data analytics and visualization.

## 🚀 Features
- KPI Dashboard (Total Load, Net Intake, Volatility, Backlog)
- Inflow vs Outflow Analysis
- Stress Detection
- Anomaly Detection
- 30-Day Forecast (ARIMA)
- Data Export (CSV + Summary Report)

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- Statsmodels

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
